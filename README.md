# djangoapp_reservation

# 概要
宿泊予約サイトです。
スタッフブログを参考に会員登録して宿泊予約をしてみよう！

# 機能・使用技術一覧
- 技術一覧
  - 言語/フレームワーク
    - Python3.7.3
    - Django2.2.4 
  - データベース
    - PostgreSQL10.10
- 機能一覧
  - ページネーション
  - カレンダー機能(django-bootstrap-datepicker-plus)
